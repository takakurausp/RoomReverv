# 部屋予約システム セットアップ手順

本システムをGoogle Apps Script (GAS) 環境へ配置し、稼働させるための手順書です。

## 1. GASプロジェクトの作成
1. お使いのGoogleアカウントでログインした状態で、[Google Apps Script](https://script.google.com/) へアクセスします。
2. 「新しいプロジェクト」をクリックし、空のプロジェクトを作成します。
3. 左上の「無題のプロジェクト」をクリックし、任意の名前（例：`部屋予約システム`）に変更します。

## 2. ファイルの転送（コピー）
ローカルに保存されている以下の4つのファイルの内容を、GASエディタ上にすべて作成します（コピー＆ペースト）。

1. **`Code.gs`**
   - 初期状態の `コード.gs` というファイルを `Code.gs` にリネーム（またはそのまま上書き）し、ローカルの `Code.js` の中身を丸ごと貼り付けて保存します。
2. **`index.html`**
   - 左側の「＋」ボタンから「HTML」を選び、`index` という名前で作成し、ローカルの `index.html` の中身を貼り付けます。
3. **`css.html`**
   - 同様に `css` という名前でHTMLファイルを作成し、`css.html` の中身を貼り付けます。
4. **`js.html`**
   - 同様に `js` という名前でHTMLファイルを作成し、`js.html` の中身を貼り付けます。

## 3. 初期設定と権限の承認（重要）
データベース（スプレッドシートの構造）の自動生成と、古い予約を自動削除するトリガーの設定、メール送信の権限許可を行います。この作業は1度だけ行います。

1. GASエディタ画面上部のメニューにある**「関数を選択」**（通常は `doGet` 等が表示されている箇所）のプルダウンをクリックします。
2. **`setupEnvironment`** を選択し、その右にある**「実行」**ボタンをクリックします。
3. **「承認が必要です」**というポップアップが表示されます。
   - 「権限を確認」をクリックします。
   - ご自身のGoogleアカウントを選択します。
   - 「Googleが確認していません」という警告が出た場合は、左下の「詳細を表示」をクリックし、「安全ではないページへ移動」をクリックします。
   - 表示されたアクセス権限（スプレッドシート、メール送信等）の内容の下にある「許可」をクリックします。
4. 画面下のコンソールに「実行完了」と表示されれば初期設定は完了です。

※この操作により、裏側で連携するスプレッドシート内に自動的に `Reservations` というデータ記録用シートが構築されます。

## 4. 部屋・ドメインのカスタマイズ（重要）
利用する部屋の追加、名前の変更、許可するメールアドレスのドメイン設定は、すべて `Code.gs` の冒頭にある定数で行います。

1. **`Code.gs`** の冒頭にある以下の部分を確認し、必要に応じて書き換えます。
   ```javascript
   // 許可するメールアドレスのドメイン（複数のドメインを許可可能）
   const ALLOWED_DOMAIN_SUFFIXES = ['@example.com'];
   // アプリのタイトル
   const APP_TITLE = '部屋予約システム';
   
   const ROOM_CONFIGS = [
     {
       name: '会議室A',
       primary: '#2563eb', // メインの色
       primaryLight: '#eff6ff', 
       bgGradient: '...グラデーション設定...',
       notice: 'ご利用上の注意を入力します'
     },
   ];
   ```
2. **ドメインの変更**: `ALLOWED_DOMAIN_SUFFIXES` を、組織のメールアドレス（`@gmail.com` など）に変更します。
3. **部屋の追加・削除**: `{ ... }` のブロックを増減させるだけで、自動的に切り替えタブが増え、カレンダーの色設定が適用されます。

## 5. Webアプリとしてデプロイ（公開）
最後の仕上げとして、システムをWebアプリとして公開します。

1. GASエディタ画面右上の青い **「デプロイ」** ボタンをクリックし、**「新しいデプロイ」** を選択します。
2. 左側の歯車アイコン（種類の選択）をクリックし、**「ウェブアプリ」** をチェックします。
3. 設定項目を以下のように指定します：
   - **説明**: 例 `初版リリース` （空欄でも可）
   - **次のユーザーとして実行**: `自分 (あなたのメールアドレス)`
   - **アクセスできるユーザー**: `全員`
4. 右下の **「デプロイ」** をクリックします。
5. 完了画面が表示され、**「ウェブアプリ」**の下に URL（`https://script.google.com/macros/s/.../exec`）が表示されます。
6. 表示された**URLをコピー**します。これがシステムの運用URL（トップページ）となります。

お疲れ様でした！このURLを皆さんに共有すればシステムの運用開始です。
