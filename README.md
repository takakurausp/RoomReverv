# RoomReserv (部屋予約システム)

Google Apps Script (GAS) と Google スプレッドシートをバックエンドに用いた、モダンで軽量な施設・部屋の予約システムです。
PC・スマートフォンの両方に最適化されたレスポンシブなUIを持ち、ユーザー登録不要でリンクを知っている関係者のみが手軽に予約システムを利用できます。

## ✨ 主な特徴 (Features)

- **ユーザー登録不要**: 専用のアカウントを作成することなく、管理者が許可したドメイン（例: `example.com`）のメールアドレスだけで予約が可能です。
- **仮予約・本予約フロー**: メール認証を介したフローにより、イタズラ予約を防止します。
- **モダンなUI/UX**: Glassmorphismを取り入れた美しく応答性の高いデザイン。PCでは月間カレンダー表示、スマホでは日ごと（カード形式）の表示に自動で最適化されます。
- **セキュアなキャンセル機能**: 予約時にワンタイムの「キャンセル専用秘密リンク」をメールで送信するため、本人以外による取り消しを防ぎます。
- **管理者向けCSV一括登録**: CSVファイルを用いて、大量の予定（講義スケジュールなど）を一瞬で登録できます。
- **完全自動のデータベース管理**: 古い予約データや、有効期限切れの仮予約は毎日深夜に自動削除され、メンテナンスフリーで運用可能です。

## 🛠️ 技術スタック (Tech Stack)

- **Backend**: Google Apps Script (GAS)
- **Database**: Google Sheets (Google スプレッドシート)
- **Frontend**: HTML5, Vanilla JavaScript, Vanilla CSS

## 🗒️ ファイル構成

- **`Code.js`**: GASバックエンド（APIエンドポイント、メール送信、スプレッドシート操作、定期削除バッチなど）
- **`index.html`**: メインのHTML構造
- **`css.html`**: デザイン定義（レスポンシブ、テーマカラー、グラスモーフィズム）
- **`js.html`**: フロントエンドロジック（カレンダー描画、バリデーション、API通信）
- **`color_tool.html`**: 部屋ごとの配色（テーマカラー）をブラウザ上でシミュレーションし、設定用コードを生成する補助ツール
- **`Setup_Guide.md`**: 構築手順書
- **`User_Manual.md`**: 運用マニュアル

## ⚙️ カスタマイズ (Customization)
`Code.js`（GAS上では `Code.gs`）の冒頭にある定数を書き替えることで、システムの挙動をカスタマイズできます。

- `APP_TITLE`: アプリ名
- `ALLOWED_DOMAIN_SUFFIXES`: 予約を許可するメールアドレスのドメイン（複数指定可）
- `ROOM_CONFIGS`: 部屋の名前、テーマカラー、各部屋特有のメッセージを定義
