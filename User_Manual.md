# 部屋予約システム ご利用マニュアル

本システムは、ブラウザから直感的に各部屋の予約状況を確認し、新規予約や取り消しが行えるツールです。
面倒なユーザーアカウント登録は一切不要で、誰でも素早くスケジュールを把握・登録できます。

---

## 1. 予約状況の確認
共有されたWebアプリのURLへアクセスすることで、即座にカレンダーを確認できます。
PC（パソコン）とスマートフォンを自動判定し、それぞれに最適化された画面が表示されます。

- **部屋の切り替え**
  画面上部のタブをクリックすると、表示する部屋を切り替えることができます。（部屋ごとにテーマカラーが変わります）

- **カレンダー表示（PCのみ）**
  PCからアクセスした場合、デフォルトで画面いっぱいの「月間カレンダー」が表示されます。予約が入っている日には、日付の下に開始時間とタイトルの先頭が表示され、空き状況がひと目でわかります。
  画面右上の「日ごと表示」ボタンで、横スクロール型の数日分カードビューに切り替えることも可能です（スマホからはこのカードビュー専用となります）。

- **特定の部屋を直接開く方法（ブックマーク向け機能）**
  URLの末尾に `?room=部屋名` のように追記してブックマーク登録しておくと、アクセスした瞬間にその部屋の画面がデフォルトで開きます。

## 2. 新規の予約を行う（仮予約と確定）
なりすまし防止のため、本システムでは「仮予約」→「メールでの確定」というステップを踏みます。
予約を行うには、管理者が許可したドメインのメールアドレスが必要です。

1. 画面右下にある青い丸の **「＋（プラス）」ボタン** をクリックします。
2. 新規予約フォームが開きます。以下の必須項目を埋めてください：
   - **日付**: 予約したい日付（本日からぴったり3ヶ月先のご予定まで指定可能）
   - **開始時間 / 終了時間**: 予約する時間帯
   - **氏名**: 予約者の名前
   - **メールアドレス**: 許可されたドメインのメールアドレスを入力
   - **目的**: 予約の用途（タイトルとして皆が見るカレンダー上に表示されます）
3. **「仮予約する」** ボタンをクリックします。（時間が重複している場合はエラーで教えてくれます）
4. **【重要】予約を確定させる**
   - 入力したメールアドレス宛てに「【（アプリ名）】... の仮予約を受け付けました」というメールが届きます。
   - メール本文内の **「▼ 予約確定用URL」** をクリックし、表示された画面で **「予約を確定する」** ボタンを押してください。
   - **15分以内** に確定操作が行われない場合、仮予約は自動的に取り消され、他の人が予約できる状態に戻ります。

## 3. 予約の取り消し・キャンセル
予約確定後、自身の予約を取り消すことができます。

1. 予約確定時（または仮予約時）のメールを開きます。
2. メール本文内にある **「▼ 確定後のキャンセル・変更について」** の下にあるリンクURLをクリックします。
3. あなた専用の「予約の取り消し確認画面」が開きます。内容をよく確認してください。
4. 赤い **「この予約を取り消す」** ボタンをクリックします。
5. 取り消しが完了し、カレンダー上からも即座に予定が削除されます。

※このキャンセル用URLはご自身専用の秘密のリンクです。絶対に他の人には共有しないようご注意ください。

## 【管理者向け機能】 CSVファイルでの一括予約
年度初めの固定スケジュールなど、大量の予定をまとめて一元登録する機能です。（この画面を利用するには管理者パスワードが必要です）

1. 画面右下の **「歯車」アイコン（管理者メニュー）** をクリックします。
2. 管理者用のパスワードを入力して認証します。
3. **「CSVテンプレートをダウンロード」** のリンクをクリックすると、一括登録専用の入力用ひな形ファイル（Excel等で編集するCSV）がダウンロードされます。
4. テンプレートの書式（Room, StartTime, EndTime, Title）に従って予定をひたすら記入し保存します。日時は `2024/04/01 09:00` のような形式で入力してください。
5. 先ほどの画面で「ファイルを選択」から作成したCSVファイルを選択し、「アップロード」ボタンを押します。
6. 完了メッセージが出れば、数百件の予約も瞬時にカレンダーへ反映されます。

**補足（システムのメンテナンス）**
- **自動お掃除機能**: 3ヶ月以上過去に過ぎ去った古い予約データは、システムが毎日深夜に自動で不要と判断し完全削除（DBの最適化）を行います。管理者が過去のデータを消す手間はかかりません。
- **手動でのデータ修正**: 万が一誤った予定を大量に一括登録してしまった場合や、どうしても特定の予約を強制的に取り消したい事情が発生した場合は、システム裏側となる **「Googleスプレッドシート（Reservationsシート）」を直接開き、該当するデータの行（番号）をまるごと削除・または書き換え** を行ってください。即座にカレンダーに結果が反映されます。
